Ecosystem Citation Index

Citation indices are tabulated based on the 625 modules in the Perl6 Ecosystem on 2016-05-01.

A total of 188 Topline Modules in the Ecosystem were cited, or 30.08% of the Ecosystem Modules.

The Citation Index is defined as 100 * # times cited by any module / # cited modules

Data is gathered from Ecosystem projects file (used by panda). A module simply cites another if it lists it in the "depends" list. A module recursively cites another if a cited module in turn cites another module.

The simple and recursive data are graphed for the top TMPL_VAR GR_NUM modules over the last TMPL_VAR N_DATES.
